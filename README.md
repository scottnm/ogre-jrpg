# OGRE JRPG :: Team 5
##### Scott Munro, Daniel Purdes, Patrick Liu

## To Build
```
cd appdirectory
./buildit
./Battle
```

## To play

* UP/DOWN/LEFT/RIGHT - navigate the menus
* ENTER - select actions
* TAB - cycle your party or targets
* ESC - quit games
